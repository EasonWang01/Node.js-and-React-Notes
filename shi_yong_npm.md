# 使用NPM

* 
先安裝Node.js

* 以下為基本指令

npm install 

npm install  -g (使用後可在cmd的任何路徑輸入package名稱執行，但如果是想在js檔內直接使用require的話，要再把環境變數加上才行)(如此即可不用在每個專案資料夾個別安裝package))
![](df.png)
npm install  --save

npm uninstall

npm search

npm ls -g

npm ls -gl

npm ls -l

npm update -g

npm update



更多可參考
