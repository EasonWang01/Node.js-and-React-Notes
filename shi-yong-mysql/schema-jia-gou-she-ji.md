# Schema 架構設計

## Messaging APP

> User: 存入 conversion Id List（參與的對話列表）
>
> Messages: 存入每個對話的細節以及 conversion Id
>
> Conversions: Key 為 conversion Id，存入該對話最後的傳輸人與時間
